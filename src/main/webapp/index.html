JEE7 Auth example

Based on: Stack Overflow: Best practice for REST token-based authentication with JAX-RS and Jersey

Using java-jwt library

With additional ideas taken from: Pairing AngularJS and Java EE 7 for authentication

Another example: AngularJS JWT Authentication Example & Tutorial

Go to TEST Page!


Good login: 
curl -v -H "Content-Type: application/json" -X POST -d '{"username":"admin","password":"secret"}' http://localhost:8080/jee7auth/rest/auth
Verify JWT Token at https://jwt.io

Bad login: 
curl -v -H "Content-Type: application/json" -X POST -d '{"username":"admin","password":"xxxx"}' http://localhost:8080/jee7auth/rest/auth

Test unsecured endpoint: 
curl -v -H "Content-Type: application/json" -X GET http://localhost:8080/jee7auth/rest/123

Test SECURED endpoint: 
curl -v -H "Content-Type: application/json" -X DELETED http://localhost:8080/jee7auth/rest/123

Test SECURED endpoint: 
TOKEN=`curl -H "Content-Type: application/json" -X POST -d '{"username":"admin","password":"secret"}' http://localhost:8080/jee7auth/rest/auth` 
curl -v -H "Content-Type: application/json" -H "Authorization: Bearer $TOKEN" -X DELETE http://localhost:8080/jee7auth/rest/123
Replace $TOKEN with your jwt token value.

RENEW token: 
TOKEN=`curl -H "Content-Type: application/json" -X POST -d '{"username":"admin","password":"secret"}' http://localhost:8080/jee7auth/rest/auth` 
curl -v -H "Content-Type: application/json" -H "Authorization: Bearer $TOKEN" -X POST http://localhost:8080/jee7auth/rest/auth/renew
